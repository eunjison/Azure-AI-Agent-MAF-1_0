## 섹션 1: AI 관련 직업 소개

### 머신러닝 엔지니어
머신러닝 엔지니어는 데이터를 이용해 AI 모델을 설계하고 학습시키는 전문가입니다.
주요 업무는 데이터 전처리, 모델 학습, 성능 평가, 배포입니다.
필요 역량: Python, TensorFlow/PyTorch, 통계학, 선형대수.
평균 연봉(국내 기준 2024): 5,000만~9,000만 원.
관련 학과: 컴퓨터공학, 인공지능학과, 전기전자공학, 수학과.

### AI 프롬프트 엔지니어
프롬프트 엔지니어는 ChatGPT, Claude 같은 대형언어모델(LLM)이 원하는 출력을 내도록
최적의 입력(프롬프트)을 설계하는 전문가입니다.
아직 국내에 신생 직군이지만 2024년 이후 수요가 급증하고 있습니다.
필요 역량: 자연어 처리 이해, 논리적 글쓰기, LLM 동작 원리 파악.

### 데이터 사이언티스트
데이터 사이언티스트는 대규모 데이터에서 의미 있는 패턴을 찾아 비즈니스 문제를 해결합니다.
업무: 탐색적 데이터 분석(EDA), 통계 모델링, 시각화, 보고서 작성.
필요 역량: Python/R, SQL, 통계학, 데이터 시각화(matplotlib, seaborn).

### AI 연구원
AI 연구원은 새로운 알고리즘과 모델을 개발하며, 대학원 진학이 일반적인 경로입니다.
국내 주요 연구소: NAVER AI Lab, Kakao AI Lab, 삼성리서치, LG AI Research.
해외: Google DeepMind, OpenAI, Meta FAIR, Microsoft Research.

### 클라우드 AI 아키텍트
클라우드 AI 아키텍트는 Azure, AWS, GCP 등 클라우드 플랫폼 위에 AI 서비스를 설계합니다.
Azure AI Services, Azure OpenAI, AI Foundry 등을 활용합니다.
필요 역량: 클라우드 인프라, 네트워크, 보안, AI/ML 파이프라인 설계.

## 섹션 2: 대학 진학 및 관련 학과 안내

### 인공지능학과
2019년 이후 국내 주요 대학에 신설된 학과입니다.
주요 대학: KAIST, 고려대, 성균관대, 한양대, POSTECH, UNIST, 연세대.
핵심 과목: 기계학습, 딥러닝, 자연어처리, 컴퓨터비전, 강화학습.

### 컴퓨터공학과
AI뿐 아니라 소프트웨어, 시스템, 네트워크 전반을 다루는 가장 범용적인 이공계 학과입니다.
AI 관련 직군 취업률이 높으며, 대부분의 상위권 대학에 개설되어 있습니다.

### 소프트웨어학과 / 소프트웨어융합학과
컴퓨터공학과 유사하지만 실무 중심 커리큘럼 비중이 높습니다.
산학협력 프로젝트, 인턴십 기회가 풍부한 경우가 많습니다.

### 수학과 / 통계학과
AI의 기반이 되는 선형대수, 미적분, 확률통계를 깊이 배웁니다.
AI 연구원 및 데이터 사이언티스트를 목표로 할 경우 탄탄한 선택지입니다.

### 입시 전략 팁
수능: 수학(미적분/기하) 필수, 과학탐구(물리/화학/정보) 권장.
생활기록부: 프로그래밍 관련 동아리, 정보올림피아드, 코딩 대회 수상 경력 유리.
포트폴리오: GitHub 프로젝트, AI 대회(Kaggle) 참여 기록이 면접에 도움.
PINE 수료증과 프로젝트 결과물도 학생부 기재 가능 여부를 담임 선생님께 확인하세요.

## 섹션 3: 과목별 학습 팁

### 수학 학습 팁
AI를 공부하려면 수학이 기초입니다. 특히 행렬(Matrix), 벡터 개념이 중요합니다.
고1: 집합, 함수, 경우의 수 → 철저히 개념 이해.
고2: 수학1(지수·로그·삼각함수), 수학2(미분·적분) → 반복 문제풀이.
고3: 확률과 통계 → 베이즈 정리, 정규분포 개념은 AI에서도 핵심.
추천 자료: 유튜브 '3Blue1Brown' (영어, 시각적 설명 탁월), 이투스/메가스터디 개념 강의.

### 영어 학습 팁
AI 논문과 공식 문서는 대부분 영어로 작성됩니다.
독해 위주로 학습하되, 기술 문서 용어(parameter, embedding, inference 등)에 익숙해지세요.
추천: 영자 뉴스(BBC, TechCrunch) 짧은 기사 매일 1개 읽기.

### 정보(컴퓨터) 과목 팁
Python 기초 → 알고리즘(정렬, 탐색) → 자료구조(스택, 큐, 트리) 순으로 학습.
정보올림피아드(KOI) 도전을 목표로 삼으면 체계적 학습에 도움이 됩니다.
Baekjoon Online Judge, Programmers에서 하루 1~2문제 꾸준히 풀기.

## 섹션 4: PINE 프로그램 FAQ

### Q: PINE 프로그램은 어떤 프로그램인가요?
PINE(P.I.N.E)은 마포청소년문화의집과 KB데이타시스템, Microsoft가 함께 운영하는
고등학생 대상 AI 에이전트 개발 프로젝트 프로그램입니다.
2026년 2기는 고등학교 1·2학년 20명이 참가하며, 총 9회기로 운영됩니다.
수료 시 디지털 인증서(배지)와 수료증이 발급됩니다.

### Q: 프로그램을 마치면 어떤 결과물이 생기나요?
팀 프로젝트로 AI 에이전트 프로토타입을 개발하고, 8회차 해커톤에서 데모 발표를 합니다.
최종 결과물은 GitHub 포트폴리오, 수료증, 디지털 배지로 입시 서류에 활용할 수 있습니다.

### Q: 프로그래밍을 잘 못해도 참여할 수 있나요?
기본적인 Python 문법(변수, 반복문, 함수)을 알면 참여 가능합니다.
2~3회차에서 Python 기초를 다시 배우며, 대학생 멘토가 팀별로 도움을 줍니다.

### Q: Azure OpenAI를 사용하려면 비용이 드나요?
PINE 프로그램 참가자에게는 실습용 API 키가 제공되므로 개인 비용이 들지 않습니다.
프로그램 종료 후 개인 실습을 원한다면 Azure 무료 크레딧($200)을 활용하세요.

### Q: 해커톤 주제는 무엇인가요?
"학교 생활 개선 에이전트"가 주제입니다.
예시: 개인화 학습 코치, 진로 상담 봇, 시험 일정 관리 에이전트 등.
팀원과 자유롭게 아이디어를 정하고, 디자인 싱킹 방법론으로 문제를 정의합니다.

### Q: Microsoft 취업이나 인턴십 기회도 있나요?
직접적인 채용 연계는 아니지만, Microsoft 직원들과 멘토링 세션이 있으며
파인콘(P.I.N.E CON) 행사에서 현직자 강연과 네트워킹 기회가 제공됩니다.

## 섹션 5: AI 에이전트 개념 정리

### 에이전트(Agent)란?
에이전트는 목표를 달성하기 위해 스스로 판단하고 행동을 선택하는 AI 시스템입니다.
단순 챗봇과의 차이: 챗봇은 질문에 답만 하지만, 에이전트는 도구를 사용하고 여러 단계를 거쳐 문제를 해결합니다.

### RAG(Retrieval-Augmented Generation)란?
RAG는 외부 문서나 데이터베이스에서 관련 정보를 검색(Retrieve)하고,
그 정보를 프롬프트에 추가해 LLM이 더 정확하고 최신의 답변을 생성(Generate)하는 방법입니다.
장점: 모델이 학습하지 않은 최신 정보도 답변에 활용 가능.
활용 예: 사내 문서 Q&A, 학교 공지 검색 봇, 법률 문서 분석.

### 멀티 에이전트(Multi-Agent)란?
여러 에이전트가 각자 다른 역할을 맡아 협력하는 구조입니다.
예: 작가 에이전트(글쓰기) + 편집자 에이전트(수정) + 검토 에이전트(사실 확인).
장점: 복잡한 작업을 분업하여 품질 향상, 단일 에이전트의 실수를 다른 에이전트가 보완.
